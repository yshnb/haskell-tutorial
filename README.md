# haskell-tutorial
Haskell tutorial for private study

## Haskellをつかえるようにする

### インストール
Macの方は以下のコマンドを実行しておいてください。
```
$ brew install ghc
```

### ghci
以下のコマンドで対話モードでHaskellがつかえます。
```
$ ghci
...
Prelude>
```

### 終了方法
```
Prelude>:quit
```
